    Introducción

    Sistema de Facturación e Inventario, SIF (Actualización: )

    En que consiste el sistema SIF, el sistema brinda las herramientas necesarias para administrar eficientemente un pequeño y mediano negocio,


    ¿Que tipos de negocios necesitan algo como SIF?


    Negocios que manejen ventas, compras, inventarios, clientes etc; por ejemplo, farmacias, supermercados, ferreterías, librerías, ventas de repuestos, restaurantes, jugueterías, almacenes, mueblerías y un largo etc.


    ¿Que partes componen el sistema SIF?


    • Clientes
      • Gestión integral de Clientes
        • Identificación mediante Fotografía
        • Limite de Crédito
        • Referencias personales
        • Archivos adjuntos (documentos de identificación, recibos de pago de agua, luz etc)

    • Productos

      • Tipos de producto

      • Existencia por Empresa
      • Historial de Movimientos
      • Advertencias sobre mínimos de existencias

        • Margen de Advertencia
      • Manejo de Vencimiento de Lotes
      • Determinación de Precios
      • Múltiples precios por producto
      • Productos Monitoreados
      • Productos/Servicios que no manejan existencias
      • Consulta de Kardex
