Compra (16/06/2024)

Las Compras tienen el objetivo de ingresar las unidades adquiridas al inventario, y por lo tanto una compra dentro del sistema es un ingreso a la existencia de un producto, se puede ingresar una compra directamente o bien se puede generar una compra con base en una orden de compra.


Ruta de acceso:


  • Menú Principal
    • Pestaña "Compras"
      • Grupo "Trasacciones"
        • "+ Compra"

Ingreso rapido de Proveedores NUEVOS


Esta opcion se usa cuando sabemos que el proveedor nunca ha sido utilizado en el sistema



O tambien cuando se ingresa/digita un NIT de proveedor que no existe en la base de datos, se lanza una pantalla para el ingreso rapido, esto ingresa los datos mas basicos (Nombre y NIT), porteriormente se debera completar los demas datos del Proveedor.
